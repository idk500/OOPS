# OOPS 项目功能清单

## 核心功能模块

### 1. 🚀 一键式检测 (One-Click Diagnostics)
- **可执行文件部署**: 提供 `oops.exe` 单文件部署，用户双击即可运行
- **图形化项目选择**: 启动时弹出窗口让用户选择项目（OneDragon、MAA、OK-WW等）
- **自动配置加载**: 根据选择的项目自动加载对应的配置模板
- **零配置启动**: 用户无需编辑YAML文件即可开始检测
- **一键修复功能**: 基于debug.bat经验的自动化问题修复

### 2. 🖥️ 硬件配置检测 (Hardware Configuration)
- **CPU兼容性检查**: 检测CPU型号，识别不支持的古典CPU（如E3等）
- **内存容量检查**: 验证内存容量是否满足项目最低要求
- **GPU能力检测**: 检查显卡型号、显存、驱动版本 >> 关键属性， 例如CUDA版本、DirectX版本查一下就好了
- **存储性能测试**: SSD/HDD识别，读写速度测试 >> RW测试TBD
- **温度监控**: CPU/GPU温度检测，过热预警 >> 这个没啥价值

### 3. 🔧 多项目支持 (Multi-Project Support)
- **集中配置管理**: 主配置文件管理所有项目启用状态
- **项目模板系统**: 预置常见项目配置模板
- **灵活检测开关**: 支持项目级和检测模块级的启用/禁用
- **配置继承机制**: 相似项目共享基础配置，减少重复

### 4. 🌐 网络连通性检测 (Network Connectivity)
- **Git仓库检测**: 主仓库和镜像仓库的连通性测试
- **PyPI源测速**: 多个PyPI镜像源的可用性和速度测试
- **项目URL检查**: 项目主页、文档、API等URL的可访问性
- **SSL证书检测**: Git SSL后端配置、系统证书完整性、网络SSL连接测试
- **并行检测引擎**: 异步并发执行，提高检测效率

### 5. 🐍 环境依赖检测 (Environment Dependencies)
- **Python环境检查**: 版本兼容性验证（3.8-3.11）
- **虚拟环境检测**: 支持venv、virtualenv、conda等多种虚拟环境
- **包依赖验证**: 自动检查requirements.txt兼容性
- **系统库检测**: CUDA、DirectX、.NET等系统级依赖 >> 与上文有重复！

### 6. 📁 路径规范检测 (Path Standards)
- **中文路径检测**: 识别并警告中文字符路径
- **权限检查**: 文件读写权限验证
- **路径长度限制**: Windows路径长度限制检查
- **特殊字符检测**: 空格、特殊字符等路径问题

### 7. 🎮 游戏设置检测 (Game Settings) - TBD
- **自动化导航**: 基于YOLO/图像识别/坐标点击的游戏设置导航
- **多模式检测**: 文本、图标、滑块、开关等多种设置类型检测
- **窗口自适应**: 处理游戏窗口大小变化的影响
- **多媒体资源管理**: 图片、模板、蒙版等资源文件组织

### 8. 🔍 智能问题诊断 (Intelligent Diagnostics)
- **历史问题匹配**: 基于知识库的智能错误模式识别
- **解决方案推荐**: 针对性问题修复建议
- **严重程度分级**: 问题优先级分类（严重、警告、提示）
- **自动化修复**: 一键修复常见问题（参考debug.bat）

### 9. 📊 报告生成系统 (Reporting System)
- **多格式报告**: HTML、JSON、文本格式报告输出
- **可视化摘要**: 检测结果图表和统计信息
- **详细日志**: 完整的检测过程记录
- **改进建议**: 具体的配置优化建议

## 基于debug.bat的修复功能

### 自动化修复项目
- **Python环境配置**: 自动设置PYTHON和PYTHONPATH环境变量
- **Git安全目录**: 自动添加项目目录到Git安全目录
- **PyPI源切换**: 支持清华源、阿里源、官方源的选择和配置
- **虚拟环境重建**: 自动重新创建损坏的虚拟环境
- **PowerShell路径**: 检查和修复PowerShell环境变量
- **SSL证书修复**: 配置Git SSL后端为schannel
- **版本检查**: 自动识别和警告旧版本
- **缓存清理**: 自动清理日志和缓存文件

## 硬件要求检测示例

### OneDragon项目硬件要求
```yaml
hardware_requirements:
  cpu:
    min_generation: 8  # 第8代Intel Core i5及以上
    excluded_models: ["E3", "E5"]  # 不支持E3等古典CPU
  memory:
    min_gb: 8
    recommended_gb: 16
  gpu:
    min_model: "GTX 1060"
    vram_min_gb: 4
    driver_min_version: "2023.01"
  storage:
    type: "SSD"  # 推荐SSD
    min_free_gb: 10
```

这个功能清单提供了项目的全面概览，后续将基于此进行详细开发和实现。
