## 📖 用户文档

位于项目根目录，面向所有用户：

| 文档 | 说明 | 适合人群 |
|------|------|----------|
| **[README.md](../README.md)** | 项目介绍、功能概览、快速开始 | 所有人 |
| **[QUICKSTART.md](../QUICKSTART.md)** | 一分钟上手指南 | 新手 |
| **[USER_GUIDE.md](../USER_GUIDE.md)** | 详细使用说明、常见问题、使用技巧 | 新手、普通用户 |
| **[USAGE.md](../USAGE.md)** | 命令行参数、配置选项、高级用法 | 高级用户、开发者 |
| **[CHANGELOG.md](../CHANGELOG.md)** | 版本更新历史 | 所有人 |

### 📝 文档关系说明

- **README.md** ← 项目入口，包含所有文档的链接
- **QUICKSTART.md** ← 最简单的入门指南，适合快速上手
- **USER_GUIDE.md** ← 详细的使用指南，包含常见问题和技巧
- **USAGE.md** ← 技术性文档，命令行参数和配置详解

**区别**：
- `QUICKSTART` vs `USER_GUIDE`：前者是快速入门，后者是完整指南
- `USER_GUIDE` vs `USAGE`：前者面向普通用户，后者面向高级用户
